About

In medical practice, monitoring mental health of a patient is as essential as its physical counterpart. This can be achieved through various established psychological assessment techniques. But they can be tedious to perform and evaluate accurately if done manually.

PsyHeal automates this using various personality insights API provided by IBM Watson under cognitive services category.
